concentration game

>>> december 2016-january 2017

Objective is to avoid contact with moving shapes and red boundary for as long as possible. Shape movement speeds up every 5 seconds.

Tools:
- Java
- Swing, Robot, MouseInfo APIs
- JUnit

** Deployed using Java Web Start through Tomcat

pixelplayer

>>> may 2016

For a data structures class, designed a Java GUI in a team of 3. Pixelplayer allows users to draw on the 9x8 matrix by selecting 1 of 4 colours, where each colour corresponds to a specific note that is played as a chord once user is finished.

Was primarily in charge of sound production and back-end, using hashtables and queues for color-sound associations.

Tools:
- Java
- Swing
